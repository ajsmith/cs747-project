# Requirements

- Python 3


# Installation

To install, run:

```shell
$ install.sh
```

This creates a Python virtual env in the directory `venv`.

Activate the environment with the command:

```shell
$ source venv/bin/activate
```


# Downloading data

Run the `download_data.sh` script.


# Parsing data

Run `cs747-parse-uniprot-fasta`
